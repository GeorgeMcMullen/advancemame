To play with AdvanceMENU in DOS and Windows you must download only one of the binary packages, unzip it in the same directory of the MAME, MESS or RAINE emulator and run the program advmenu. For other operating systems you must download the source package and compile it.

Please consider to support the Advance Project with a Donation.

FileDescription
advancemenu-2.8.tar.gz The sources for GNU/Linux, Mac OS X, DOS and Windows
advancemenu-2.8-dos-x86.zip DOS binary for x86
advancemenu-2.8-windows-x86.zip Windows binary for x86

You can download all the previous versions from SourceForge.

The DOS binaries are compiled with the gcc-3.2.3 compiler using the djdev 2.03 environment under GNU/Linux. The Windows binaries are compiled with the gcc-4.8.1 compiler using the MingW environment under GNU/Linux.

Snapshot

AdvanceMENU can use static snapshot images PNG/PCX/ICO, animated shapshot clips MNG and sounds WAV/MP3. They aren't required but are very useful to remember games.

The following is my collection of of MP3/MNG files that it's mostly derived from the CrashTest collection with some addition.

Note that this is a collection of .MNG files, and not of .AVI files. To see them, you need a a program supporting this format, like AdvanceMENU.
