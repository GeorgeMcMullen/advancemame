To use the Advance Projects in DOS and Windows you have to download one of the binary packages. For Linux, Mac OS X and other platforms you must download the source package and compile it.

Good starting points are the build.txt and install.txt documentation files.

You can download all the previous versions from SourceForge.

The DOS binaries are compiled with the gcc-3.2.3 compiler using the djdev 2.03 (refresh) environment under GNU/Linux. The Windows binaries are compiled with the gcc-4.8.1 compiler using the MingW environment under GNU/Linux.

License

The sources of the AdvanceMAME/MESS projects are released under the GNU GPL license with a special exception that allows the link with the MAME code.

Please note that to use the AdvanceMAME/MESS binaries linked with the original MAME sources you must also follow the MAME License.

More details are in the COPYING file present in all the archives.
