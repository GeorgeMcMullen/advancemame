How to get support for you problems.

Self-Help Resources

Before contacting the project users and developers regarding a support issue, we encourage you to try using any self-help materials that the project provides, such as documentation, known bug lists, and troubleshooting guides.

The most likely places to find information of this nature is the documentation included in the file releases and the advancemame-user mailinglist and forum archives.

You can use the following box to search the documentation present in this web site. The mailing list and forum archives have their search engine.

Support Resources

You can contact the other project users using the Forums and ask for help.

If you have found a bug you can fill a Bug Report.
